# Collaborative Code Editor Platform

A scalable, real-time web application built using a modern microservices architecture. This platform enables live code editing with seamless real-time communication.

---

## 📌 Features

- ⚡ Real-time collaboration using WebSockets (Socket.io)
- 🧠 In-browser code editing powered by Monaco Editor
- 🔄 Live updates across multiple clients
- 🏗️ Microservices-based architecture
- 🐳 Containerized using Docker
- ☁️ Cloud deployment on AWS ECS
- 📈 Highly scalable and fault-tolerant system design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Monaco Editor

### Backend
- Node.js
- Express.js

### Real-Time Communication
- Socket.io (WebSockets)

### DevOps & Deployment
- Docker
- AWS ECS

### Architecture
- Microservices
- Scalable System Design
